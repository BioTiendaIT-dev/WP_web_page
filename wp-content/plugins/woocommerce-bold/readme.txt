## Descripción

El plugin WooCommerce Bold Payment Gateway permite la integración de tu tienda en línea de WordPress con la pasarela de pagos de Bold Colombia. Este plugin proporciona a tus clientes una opción segura y eficiente para realizar pagos en línea.

### Características Destacadas

- Integración con la pasarela de pagos de Bold Colombia.
- Personalización de la apariencia de los botones de pago.
- Actualización automática del estado de las órdenes mediante funciones cron.
- Soporte para la moneda colombiana (COP).

## Requisitos

- WordPress 4.0 o superior.
- WooCommerce 2.0.0 o superior.

## Instalación

1. Descarga el archivo zip del plugin desde [enlace de descarga].
2. Ve a tu panel de administración de WordPress.
3. Navega a `Plugins` > `Añadir Nuevo`.
4. Haz clic en `Subir plugin` y selecciona el archivo zip descargado.
5. Activa el plugin una vez que se haya completado la instalación.

## Configuración

1. Ve a `WooCommerce` > `Ajustes` > `Pagos` > `Bold`.
2. Activa la opción "Habilitar Bold Payments".
3. Completa los campos requeridos con las credenciales de API y llaves secretas para producción y pruebas.
4. Configura otras opciones según tus preferencias.

## Uso del Plugin

1. Configura los ajustes de Bold en la sección correspondiente.
2. Asegúrate de que el plugin esté habilitado.
3. Realiza una transacción de prueba para verificar la integración.
